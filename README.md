# Z&Z Assessoria de Trânsito 🚦

Este repositório contém o código-fonte da landing page oficial da **Z&Z Assessoria**, desenvolvida para o especialista **Richard Zawitoski**. O projeto foi concebido com uma estética "Dark & Gold", focando em autoridade, sofisticação e conversão de leads para serviços jurídicos de trânsito em Mauá e região.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores tecnologias para performance e design moderno:

* **HTML5 & CSS3** (Estrutura e semântica)
* **Tailwind CSS** (Framework de estilização utilitária para layout responsivo)
* **Font Awesome** (Ícones profissionais e temáticos)
* **AOS (Animate On Scroll)** (Biblioteca de animações de deslize de alta performance)
* **Google Fonts** (Tipografia Montserrat para leitura clara e elegante)

## ✨ Diferenciais do Projeto

* **Design Premium:** Paleta de cores Dark & Gold com estética industrial.
* **Experiência Temática:** Elementos visuais que remetem ao trânsito (fitas zebradas, cones e animações de movimento).
* **Mobile-First:** Otimização total para navegação via smartphone.
* **Social Proof:** Integração visual de avaliações reais do Google com logotipia oficial.
* **Efeitos de Scroll:** Animações fluídas que conferem o aspecto de "site de alto valor".

## 🚀 Como Visualizar

O projeto está configurado para deploy imediato. 

1.  Clone este repositório.
2.  Abra o arquivo `index.html` em qualquer navegador ou utilize a extensão Live Server do VS Code.
3.  O deploy oficial é gerenciado via **Vercel**.
